# Evaluation Rubric

Use this rubric to evaluate whether a generated continuation follows a Work DNA profile.

## Score table

Score each dimension from 1 to 5.

| Dimension | 1 | 3 | 5 |
|---|---|---|---|
| Canon continuity | Contradicts key facts | Mostly consistent, minor gaps | Fully consistent and uses unresolved threads |
| POV/focalization | Frequent leaks or wrong distance | Mostly right, occasional drift | Stable and source-consistent |
| Narrative engine | Generic events | Some source-like tension | Same page-turning mechanism as source |
| Scene architecture | Wrong opening/escalation/ending | Some matching beats | Clearly preserves source scene mechanics |
| Language texture | Generic prose | Some matching rhythm | Sentence, diction, imagery, and dialogue patterns align |
| Character grammar | Characters behave out of pattern | Mostly plausible | Decisions, speech, and avoidance patterns match |
| Information control | Reveals too much/too little | Some pacing issues | Withholding/reveal pattern matches source |
| Requirement satisfaction | Misses user request | Partially satisfies | Satisfies without breaking DNA |

## Repair instruction patterns

Use specific repair verbs:

- Replace direct emotion labels with bodily action.
- Delay the explanation until after the sensory clue.
- Reduce narrator knowledge to what the focal character can perceive.
- Split exposition across action beats.
- Restore the unresolved thread from the previous scene.
- Shorten dialogue turns and add evasive subtext.
- Re-anchor imagery in the source's motif family.
